Supported CI

CI tools support is still under testing. Please help to report issue if you encounter any problem.

GitHub Action

Below is a step example:

- name: Report
  uses: covergates/github-actions@v1
  with:
    report-id: "Report ID"
    report-type: "go"
    report-file: "./coverage.out"
    pull-request: "true"

Arguments:

  • report-id Report ID could be found in repository setting
  • report-type refer to Upload Command for more detail
  • report-file coverage report
  • pull-request set true to leave a comment on your pull request
Reach the source code for more detail.
